# Catalog Manager

A modern product management dashboard built with React, TypeScript, and Tailwind CSS.

## ✨ Features

- Product listing with pagination
- Search with debounce
- Add new product
- Responsive dashboard layout
- Clean and modern UI

## 🛠 Tech Stack

- React
- TypeScript
- Tailwind CSS
- Axios
- Lucide React

## 🚀 Getting Started

```bash
npm install
npm run dev
```
